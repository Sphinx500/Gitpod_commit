¿Qué son los Commit Docker?

Crea una nueva imagen a partir de los cambios de un contenedor.

Puede resultar útil confirmar los cambios o la configuración del archivo de un contenedor en una nueva imagen. Esto le permite depurar un contenedor ejecutando un shell interactivo o exportar un conjunto de datos de trabajo a otro servidor. Generalmente, es mejor usar Dockerfiles para administrar sus imágenes de una manera documentada y mantenible. Obtenga más información sobre los nombres y etiquetas de imágenes válidos. La operación de confirmación no incluirá ningún dato contenido en los volúmenes montados dentro del contenedor.
